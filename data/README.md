# Data

This project uses the Airline Passenger Satisfaction dataset from Kaggle.

The dataset contains 129,880 passenger survey records, including demographic information, flight characteristics, service quality ratings, and overall satisfaction labels.

Features include:

* Demographic Attributes

  * Gender
  * Age

* Flight Characteristics

  * Customer Type
  * Type of Travel
  * Class
  * Flight Distance

* Service Ratings

  * Inflight WiFi Service
  * Online Boarding
  * Seat Comfort
  * Food and Drink
  * Leg Room Service
  * Baggage Handling
  * Inflight Entertainment
  * Departure and Arrival Delays

Target Variable:

* Satisfaction

  * Satisfied
  * Neutral/Dissatisfied

The raw dataset is publicly available through Kaggle and is not redistributed in this repository.
